# Microservice zur Verwaltung von Zuliefererdaten

Dieses Projekt umfasst die Entwicklung und das Deployment eines Spring Boot Microservices zur Verwaltung
von Zuliefererdaten. Der Microservice bietet sowohl REST als auch GraphQL Schnittstellen, um
CRUD-Operationen (Create, Read, Update, Delete) zu ermöglichen. Er wurde als Docker-Container erstellt
und als Pod in einem Kubernetes-Cluster deployt, um eine skalierbare und containerisierte Architektur zu
gewährleisten.
